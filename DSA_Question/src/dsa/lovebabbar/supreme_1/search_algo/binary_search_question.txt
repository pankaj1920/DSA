Easy:

	1- Find the index of a given element in a sorted array.
	2- Determine if a given number is present in a sorted array.
	3- Calculate the square root of a non-negative integer using binary search.
	4- Find the peak element in an array (an element greater than its neighbors).
	5- Search in a rotated sorted array to find a target element.
	6- Find the first occurrence of a target element in a sorted array.
	7- Find the last occurrence of a target element in a sorted array.
	8- Search for an element in a nearly sorted array.
	9- Count occurrences of a target element in a sorted array.
	10- Search a 2D matrix (sorted row-wise and column-wise) for a target element.
	11- Find the maximum element in a bitonic array.
	12- Search for a target element in a mountain array.
	13- Search for an element in an infinite sorted array.
	14- Search for a target element in a circularly sorted array.
	15- Find the element just smaller than the target in a sorted array.

Medium:

	1- Find the minimum element in a rotated sorted array.
	2- Search for the "kth" smallest element in a sorted matrix.
	3- Find the median of two sorted arrays.
	4- Find the smallest missing positive integer in an array.
	5- Find the element that appears only once in a sorted array (others appear twice).
	6- Find the index of the peak element in an array.
	7- Search for a target element in a sorted array with unknown size.
	8- Search for a target range in a sorted array.
	9- Find the smallest element greater than a given target in a sorted array.
	10- Find the smallest element larger than a given target in a rotated sorted array.

Hard:

	1- Find the "kth" element in two sorted arrays.
	2- Search in a rotated sorted array with duplicates.
	3- Search in a sorted, but rotated, array containing duplicates.
	4- Allocate minimum number of pages for books to be read by students.
	5- Search in a sorted array of strings which is interspersed with empty strings.
